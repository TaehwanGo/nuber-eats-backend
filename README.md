# Nuber Eats

The Backend of Nuber Eats Clone

## User Entity:

- id
- createAt
- updatedAt

- email
- password
- role( client | owner | delivery )
  - client: restaurant list
  - owner: dash board
  - delivery: realtime time order list

## User CRUD:

- Create Account
- Log In
- See Profile
- Edit Profile
- Verify Email
